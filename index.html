Issue #1

No Meta Descriptions
On Any Page

Every single page crawled — homepage, services, about, sector pages — is missing a meta description. Google will auto-generate snippets from page content, which are often poor representations of the page's value.

Evidence

/ — meta description: (none)
/about-us/ — meta description: (none)
/what-we-do/salary-packaging/ — (none)
/what-we-do/novated-leasing/ — (none)
/who-we-help/not-for-profits/ — (none)
✓
Recommended Fix

Write unique, keyword-rich meta descriptions (150-160 chars) for every page. Prioritise the homepage, salary packaging, novated leasing, and sector pages first.

04 / 11
Critical

Issue #2

Zero Social
Sharing Tags

No Open Graph tags and no Twitter Card tags on any page. When shared on LinkedIn, Facebook, or Twitter, the site has no control over the title, description, or image that appears.

No og:title, og:description, og:image

Social platforms will guess at content to display, often pulling irrelevant text or no image at all. For a B2B company, LinkedIn shares are critical.

No Twitter Card meta tags

twitter:card, twitter:title, twitter:image — all missing. Twitter/X shares will appear as plain text links.

✓
Recommended Fix

Add og:title, og:description, og:image, og:url, og:type + twitter:card tags to every page template. Use branded images (1200×630px) for consistent social previews.

06 / 11
High

Issue #4

Soft 404s &
Missing Alt Text

Old URLs return a 200 status with "404" content — confusing crawlers. And 4 of 10 homepage images have empty alt attributes.

Soft 404s on legacy URLs

/salary-packaging/ and /novated-leasing/ display "404" content but return HTTP 200. Google treats these as live pages with thin content — hurting crawl quality.

40% of homepage images missing alt

The hero slider images, logo image, and carousel images all have empty alt="" attributes. Impacts accessibility and image search visibility.

✓
Recommended Fix

Return proper 404 status codes (or 301 redirect to /what-we-do/salary-packaging/). Add descriptive alt text to all images.

08 / 11

Medium Priority

Other Findings

  • BreadcrumbList schema malformed — uses "context" and "type" instead of the required "@context" and "@type". May not be parsed correctly by Google.
  • No Organization schema — an ASX-listed company should have complete Organization markup with logo, address, social profiles, and contact info.
  • Title tags are generic — "Salary Packaging | Smartgroup" misses keywords like "Australia", "employee benefits", or "for employers".
  • Viewport meta incomplete — only has "width=device-width" (missing "initial-scale=1").
  • Sitemap contains stale entries — many news articles have identical lastmod dates from March 2024, suggesting the sitemap isn't regenerating properly.
